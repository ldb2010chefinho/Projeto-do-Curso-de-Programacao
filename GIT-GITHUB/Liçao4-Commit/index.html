Git: Sistema de controle de versão distribuído, criado por Linus Torvalds (mesmo criador do Linux). Ele permite que você acompanhe todas as mudanças feitas no código ao longo do tempo, crie versões paralelas (branches), volte atrás se algo der errado e colabore com outras pessoas.

Melhores codigos do Git(Minha opnião):

  • git status- Mostra o estagio atual do projeto;
  • git add- Adiciona as atuais mudanças do codigo na staging area;
  • git init- Inicializa o repositorio git do projeto.
